## ğŸ§½ Detail Pembersihan

### Quick Clean (5 langkah)
Cocok untuk pembersihan rutin sehari-hari:
- Temporary files user
- Temporary files Windows
- Prefetch
- Recent files
- Recycle Bin semua drive

### Deep Clean (15 langkah)
Pembersihan menyeluruh, cocok dilakukan seminggu sekali:
- Semua yang ada di Quick Clean
- DNS Cache
- Cache Chrome, Edge, Firefox
- Windows Update Cache
- Thumbnail Cache
- Log Files
- Error Reports
- Delivery Optimization Cache
- Windows Defender Cache

### Custom Clean
Pilih sendiri area yang ingin dibersihkan:
- Temporary Files
- Browser Cache (semua browser)
- Windows Update Cache
- Recycle Bin
- Thumbnail Cache
- Log Files
- DNS Cache
- Prefetch

### Analisa Sampah
Cek dulu berapa besar file sampah di komputer kamu sebelum memutuskan untuk membersihkan.
